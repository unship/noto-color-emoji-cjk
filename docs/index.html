Top: Noto color emoji. Bottom: symbols merged from Symbola (enclosed alphanumerics, mahjong, dominoes, cards, arrows) — all on the same grid. Color requires a browser with sbix support (Safari); the cell alignment is identical everywhere.

Why

  • Emacs's macOS NS build paints only sbix color fonts — so Noto's PNG strikes are repackaged as sbix.
  • Every glyph is re-composited onto the donor CJK font's cell, so emoji align to the 2-cell grid at every size.
  • Non-emoji symbols (like 🅔 U+1F154) are filled from Symbola as crisp vector glyphs, scaled and centered onto the same cell.

Install

brew tap unship/fonts
brew trust unship/fonts
brew install --cask font-noto-color-emoji-cjk

Use in Emacs

;; Primary: an equal-width CJK mono font (1:2 cell grid)
(set-face-attribute 'default nil :family "Maple Mono NF CN" :height 140)

;; Route emoji + symbols to the metrics-matched companion (no :size).
(let ((f (font-spec :family "Noto Color Emoji CJK")))
  (set-fontset-font t 'emoji nil)
  (set-fontset-font t 'emoji f)
  (set-fontset-font t '(#x1F000 . #x1FFFF) f))

It's a companion font: keep your monospace CJK font as the primary and use this as the emoji/symbol fallback. Tune it to your CJK font by rebuilding with that font as the metrics donor — see the repo.
